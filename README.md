# README

Progetto contenente estensioni chorus:

* activities: alcuni tipi di activity sono implementati in questo progetto in modo da poter essere
selettivamente incluse nelle versioni opportune dei building-blocks.
* Il progetto dipende da Chorus in quanto ne utilizza alcuni tipi base.
* In un prossimo futuro &egrave; possibile che le activit&agrave; chorus siano spostate dal progetto
base e vengano inserite in progetti satellitari con il nome di tpm-chorus-(x | y | c | .. )
